# USB Formatting Using DiskPart

## Overview

This project demonstrates how to prepare and configure a USB drive for different use cases including Windows compatibility, macOS compatibility, and bootable media creation. It focuses on file system selection and DiskPart usage.

---

## Recommended Usage

### NTFS
- Windows system and internal drives  
- Environments requiring permissions and security  

### exFAT
- Large USB storage devices  
- Cross-platform file transfer (Windows, macOS, Linux)  
- Files larger than 4 GB  

### FAT32
- USB flash drives  
- Bootable media (UEFI and legacy systems)  
- Maximum compatibility across devices
